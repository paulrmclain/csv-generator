Winnie, the CSV Generator
About

This game is inspired by Wordle. You will have five tries to guess the five letter word.

This application was written in Python, using Flask as the framework and a mixture of HTML and JQuery for the frontend. It runs in Google App Engine, a serverless platform within Google Cloud Platform.

Refresh to get a new random word, or visit a specific puzzle by appending the number to the URL. There are approximately 5,700 words available.

Last updated on January 6, 2023.
